## 个人博客
### 主要是练手，还有地方需要完善，这里先这样，有空再做吧。主要功能：后台的增删改查，以及前台页面的渲染，主要用到技术如下：
* tp5，经典的MVC框架
* vue框架，前端渲染
* vue的UI框架MintUI
* bootstrap

### 运行项目时，public为项目根路径哦，apache不多说，nginx要记得配置一下，让index.php不显示，不然会报错的(⊙o⊙)哦。
